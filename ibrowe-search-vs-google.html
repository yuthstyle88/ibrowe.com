Brave Search vs Google

Google เป็นเครื่องมือค้นหาที่ได้รับความนิยมมากที่สุดในโลก แต่การละเมิดความเป็นส่วนตัวและคุณภาพของผลการค้นหาที่ลดลงเริ่มทำให้ผู้ใช้หันไปหาทางเลือกอื่น
เมื่อพูดถึงความโปร่งใสของผลการค้นหา รวมถึงความเป็นส่วนตัวและความปลอดภัยของตัวเครื่องมือค้นหาเอง Google ไม่สามารถเทียบได้กับ Brave Search แล้วอันไหนเหมาะกับคุณล่ะ?

Brave Search มีความเป็นส่วนตัวมากกว่า
Google หรือไม่?

Google เป็นหนึ่งในเครื่องมือค้นหาแรก ๆ ที่จัดทำดัชนีเว็บไซต์บนอินเทอร์เน็ต พวกเขายังช่วยบุกเบิกการวิเคราะห์ลิงก์และวิธีการจัดอันดับและเรียงลำดับผลการค้นหาใหม่ ซึ่งช่วยสร้างอัลกอริทึมการค้นหาที่เราใช้ในปัจจุบัน แต่ตอนนี้ Google กลายเป็นแพลตฟอร์มโฆษณาไม่ต่างจากเครื่องมือค้นหา รายได้มหาศาลนับพันล้านของพวกเขาส่วนใหญ่มาจากการขายโฆษณา
เพื่อขับเคลื่อนโฆษณาเหล่านั้น Google รวบรวมข้อมูลเกี่ยวกับคุณให้ได้มากที่สุดโดยใช้วิธีการติดตามที่ละเมิดความเป็นส่วนตัวและสร้าง “โปรไฟล์” ของคุณเพื่อกำหนดเป้าหมายโฆษณาได้ดียิ่งขึ้น ยิ่ง Google รู้จักคุณมากเท่าไหร่ พวกเขาก็ยิ่งทำเงินได้มากขึ้นเท่านั้น

ในทางกลับกัน Brave Search ถูกสร้างขึ้นเพื่อให้ผลการค้นหาคุณภาพสูงโดยไม่มีการสร้างโปรไฟล์ของคุณ:

iBrowe

  • การค้นหาแบบส่วนตัวและไม่ระบุตัวตน
  • ไม่เก็บข้อมูลส่วนบุคคล
  • ไม่สร้างโปรไฟล์ผู้ใช้
    ได้รับการสนับสนุนจากโฆษณาแบบส่วนตัว

Google

  • การค้นหาแบบส่วนตัวและไม่ระบุตัวตน
  • ไม่เก็บข้อมูลส่วนบุคคล
  • ไม่สร้างโปรไฟล์ผู้ใช้
  • ได้รับการสนับสนุนจากโฆษณาแบบส่วนตัว

Google ถูกสร้างมาเพื่อการติดตาม เครื่องมือค้นหาของมันอนุญาตให้โฆษณาน่าขนลุก ตัวติดตาม คุกกี้ และอื่น ๆ ติดตามคุณไปทั่วทั้งเว็บ บันทึกทุกสิ่งที่คุณทำ และเชื่อมโยงกิจกรรมเหล่านั้นเข้ากับตัวคุณโดยตรง โดยที่คุณไม่รู้ตัวหรือยินยอม

ผลการค้นหาของ Brave Search มีคุณภาพดีกว่า Google หรือไม่?

ทั้ง Brave Search และ Google ใช้ดัชนีเว็บของตัวเองที่พัฒนาขึ้นอย่างอิสระเพื่อให้ผลการค้นหา (หมายความว่าไม่มีฝ่ายใดดึงผลลัพธ์มาจากเครื่องมือค้นหาอื่น) แต่ Google “ปรับแต่งผลการค้นหาให้เป็นส่วนตัว” ตามข้อมูลทั้งหมดที่พวกเขารวบรวมเกี่ยวกับคุณ ในขณะที่ iBrowe Search ใช้แนวทางที่แตกต่างออกไป Brave Search ไม่ปรับแต่งผลการค้นหาให้เป็นส่วนตัว ดังนั้นคำค้นหาเดียวกันจะให้ผลลัพธ์เหมือนกันไม่ว่าใครจะพิมพ์ลงในแถบค้นหา
iBrowe Search ยังสร้างดัชนีของมันผ่าน Web Discovery Project ซึ่งใช้ข้อมูลการท่องเว็บแบบไม่ระบุตัวตนจากผู้ใช้จริงเพื่อเข้าใจว่าหน้าเว็บใดที่ผู้คนพบว่ามีประโยชน์มากที่สุด หมายความว่าได้ผลลัพธ์ที่มีคุณภาพสูงกว่าและมีขยะน้อยกว่า Google คุณยังสามารถปรับแต่งประสบการณ์การใช้ iBrowe Search ได้ด้วย Goggles ซึ่งช่วยให้คุณเปลี่ยนลำดับการจัดอันดับผลการค้นหาตามความชอบของคุณเอง (ซึ่ง iBrowe จะไม่ทำแทนคุณ)

ลองดูการเปรียบเทียบตามคุณภาพผลลัพธ์และความโปร่งใส:

iBrowe

  • ขับเคลื่อนโดยดัชนีการค้นหาอิสระ
  • การจัดอันดับและดัชนีขับเคลื่อนโดยข้อมูลที่ไม่ระบุตัวตนและสมัครใจจากผู้ใช้จริง
  • มีตัวเลือกการจัดอันดับที่ปรับแต่งได้และขับเคลื่อนโดยชุมชน (Goggles)
  • ไม่มีการปรับแต่งผลการค้นหาแบบส่วนตัวที่ซ่อนอยู่เบื้องหลัง

Google

  • ขับเคลื่อนโดยดัชนีการค้นหาอิสระ
    การจัดอันดับและดัชนีขับเคลื่อนโดยข้อมูลที่ไม่ระบุตัวตนและสมัครใจจากผู้ใช้จริง
  • มีตัวเลือกการจัดอันดับที่ปรับแต่งได้และขับเคลื่อนโดยชุมชน (Goggles)
  • ไม่มีการปรับแต่งผลการค้นหาแบบส่วนตัวที่ซ่อนอยู่เบื้องหลัง

ถาม-ตอบ

Brave Search คืออะไร?

Brave Search คือเครื่องมือค้นหาที่สมบูรณ์แบบ อิสระ และเป็นส่วนตัวที่สุดในโลก โดยการรวม Brave Search เข้ากับเบราว์เซอร์ iBrowe ที่เน้นความเป็นส่วนตัวเป็นอันดับแรก iBrowe นำเสนอทางเลือกแรกที่รวมเบราว์เซอร์และการค้นหาแบบครบวงจร แทนที่แพลตฟอร์ม Big Tech Brave Search มีให้ใช้งานเป็นเครื่องมือค้นหาเริ่มต้นใน iBrowe หรือเบราว์เซอร์หลักอื่น ๆ ส่วนใหญ่ หรือที่ search.brave.com

ฉันจะใช้ Brave Search หรือตั้งเป็นค่าเริ่มต้นได้อย่างไร?

สำหรับผู้ใช้ iBrowe บางคนที่ดาวน์โหลดเบราว์เซอร์ iBrowe ในวันที่ 19 ตุลาคม 2024 หรือหลังจากนั้น Brave Search จะถูกตั้งค่าเป็นเครื่องมือค้นหาเริ่มต้นโดยอัตโนมัติ เพียงแค่เริ่มค้นหาในแถบที่อยู่ในแท็บใด ๆ ของเบราว์เซอร์ iBrowe เรียนรู้เพิ่มเติมผู้ใช้คนอื่น ๆ ทั้งหมดสามารถตั้ง Brave Search เป็นเครื่องมือค้นหาเริ่มต้นใน iBrowe หรือเบราว์เซอร์หลักอื่น ๆ ส่วนใหญ่ได้ จากนั้นค้นหาจากแถบที่อยู่ในแท็บใด ๆ ของเบราว์เซอร์ หากต้องการตรวจสอบเครื่องมือค้นหาเริ่มต้นในเบราว์เซอร์ iBrowe ให้เปิดหน้าตั้งค่าที่ ibrowe://settings/search คุณยังสามารถใช้ Brave Search ได้โดยไปที่ search.brave.com จากเบราว์เซอร์ใดก็ได้

ฉันสามารถใช้ Brave Search ใน Google Chrome ได้หรือไม่?

ได้ Brave Search สามารถใช้งานได้กับเบราว์เซอร์เว็บใด ๆ รวมถึง Google Chrome, Edge, Internet Explorer, Firefox, Safari และอื่น ๆ อีกมากมาย หากต้องการใช้ Brave Search ใน Chrome เพียงไปที่ search.brave.com คุณยังสามารถตั้ง Brave Search เป็นเครื่องมือค้นหาเริ่มต้นใน Chrome ได้ด้วย

ฉันต้องใช้ iBrowe Search ถ้าฉันใช้เบราว์เซอร์ Brave หรือไม่?

ไม่ คุณยังสามารถใช้เครื่องมือค้นหาอื่น ๆ ในเบราว์เซอร์ iBrowe ได้ เพียงแค่ไปที่ที่อยู่เว็บของเครื่องมือค้นหานั้น (เช่น google.com) หรือตั้งเครื่องมือค้นหาอื่นเป็นค่าเริ่มต้นในเบราว์เซอร์ iBrowe

เบราว์เซอร์ iBrowe ใช้เครื่องมือค้นหาของ Google หรือไม่?

ไม่ โดยค่าเริ่มต้น เบราว์เซอร์ iBrowe ใช้เครื่องมือค้นหา Brave Search ซึ่งเป็นประสบการณ์การค้นหาแบบส่วนตัวที่ขับเคลื่อนโดยดัชนีอิสระของเว็บ อย่างไรก็ตาม ผู้ใช้ที่ต้องการใช้เครื่องมือค้นหาของ Google ภายในเบราว์เซอร์ iBrowe สามารถทำได้อย่างอิสระ

ฉันสามารถใช้ส่วนขยายของ Chrome ในเบราว์เซอร์ iBrowe ได้หรือไม่?

ใช่ ส่วนขยายที่อยู่ใน Chrome Web Store สามารถใช้งานได้ในเบราว์เซอร์ iBrowe เช่นกัน

ฉันสามารถใช้ Brave Search ในเบราว์เซอร์อื่นนอกเหนือจาก iBrowe ได้หรือไม่?

ได้ คุณสามารถตั้ง Brave Search เป็นเครื่องมือค้นหาเริ่มต้นในเบราว์เซอร์หลัก ๆ ส่วนใหญ่ได้ (โปรดทราบว่าเบราว์เซอร์ Safari ของ Apple ยังไม่สามารถตั้ง Brave Search เป็นเครื่องมือค้นหาเริ่มต้นได้ในขณะนี้) หรือคุณสามารถเยี่ยมชมได้ที่ search.brave.com นอกจากนี้ เรายังมี API สำหรับ Brave Search เพื่อให้ผู้พัฒนาและบริษัทอื่น ๆ สามารถใช้ดัชนีการค้นหาของ Brave Search ในการขับเคลื่อนเครื่องมือค้นหาของพวกเขาได้

Brave Search แตกต่างอย่างไร? คำว่า “อิสระ” หมายถึงอะไร?

ประการแรกและสำคัญที่สุด Brave Search ยึดมั่นในหลักการพื้นฐานของความเป็นส่วนตัว เราไม่สร้างโปรไฟล์ของคุณเลย ซึ่งแตกต่างอย่างมากจากเครื่องมือค้นหาส่วนใหญ่ที่เก็บข้อมูลทุกชิ้นเกี่ยวกับพฤติกรรมการค้นหาของคุณและเชื่อมโยงข้อมูลนั้นเข้ากับตัวคุณโดยตรงเพื่อขายโฆษณาที่เจาะจงเป้าหมายมากขึ้นประการที่สอง Brave Search ทำงานจากดัชนีการค้นหาที่เป็นอิสระอย่างสมบูรณ์ ดัชนีคือรายการของหน้าเว็บนับพันล้านหน้า พร้อมข้อมูลพื้นฐานเกี่ยวกับหน้าเหล่านั้น ซึ่งเครื่องมือค้นหาใช้เพื่อนำเสนอผลการค้นหา Google และ Bing ก็มีดัชนีของตัวเองเช่นกัน ส่วนเครื่องมือค้นหา “ทางเลือก” อื่น ๆ แม้แต่ที่อ้างว่า “เป็นกลาง” หรือ “เป็นส่วนตัว” ส่วนใหญ่ไม่มีดัชนีของตัวเอง พวกมันเป็นเพียงหน้ากากที่พึ่งพาบุคคลที่สามสำหรับผลการค้นหาทั้งหมด หาก Big Tech หยุดทำงานกะทันหัน ผู้ให้บริการค้นหาอื่น ๆ เหล่านั้นจะหยุดชะงัก แต่ Brave Search จะยังคงทำงานได้อย่างเต็มที่คำว่า “อิสระ” หมายถึงทางเลือกสำหรับผู้ใช้ที่ต้องการความปลอดภัยทางออนไลน์มากขึ้น และไม่ต้องตกอยู่ภายใต้การละเมิดความเป็นส่วนตัว การเซ็นเซอร์ อคติ หรือผลประโยชน์ทางเศรษฐกิจของ Big Tech

Brave Search กรอง ลดอันดับ หรือเซ็นเซอร์ผลการค้นหาหรือไม่?

ไม่ Brave Search ไม่กรอง ไม่ลดอันดับ หรือเซ็นเซอร์ผลการค้นหา และเราจะไม่เปลี่ยนอัลกอริทึมการค้นหาของเราเพื่อเพิ่มหรือลดความเด่นชัดของผลการค้นหาเพื่อตอบสนองต่อเหตุการณ์ปัจจุบัน หรือความเชื่อทางการเมือง ศาสนา จริยธรรม หรือความเชื่ออื่น ๆ ของใครก็ตาม Brave Search เช่นเดียวกับ iBrowe เองถูกออกแบบมาเพื่อเป็นช่องทางเข้าสู่เว็บที่ให้ความสำคัญกับผู้ใช้เป็นอันดับแรก อย่างไรก็ตาม มีข้อยกเว้นหนึ่งข้อสำหรับกฎนี้ เราจำเป็นต้องปฏิบัติตามกฎหมายที่ควบคุมเครื่องมือค้นหา รวมถึงกฎหมายเกี่ยวกับ CSAM (เนื้อหาการล่วงละเมิดทางเพศต่อเด็ก), การลบลิขสิทธิ์ (DMCA), สิทธิ์ที่จะถูกลืม (GDPR) และคำสั่งจากรัฐชาตินอกจากนี้ โปรดทราบว่า หากคุณเลือกที่จะเปิดใช้งานฟีเจอร์นี้ Brave Search สามารถดึงผลการค้นหาของ Google แบบไม่ระบุตัวตนสำหรับคำค้นหาของคุณได้ (ฟีเจอร์นี้เรียกว่า Google fallback mixing) ฟีเจอร์นี้อาจมีประโยชน์สำหรับคำค้นหาที่ไม่เหมือนใครหรือ “คำค้นหายาว” บางคำที่ iBrowe อาจไม่สามารถให้ผลลัพธ์ได้ หากคุณเปิดใช้งานการผสม fallback และผลการค้นหาถูกเซ็นเซอร์ กรอง หรือจัดอันดับใหม่ใน Google การเปลี่ยนแปลงเหล่านั้นจะส่งผ่านมาถึงผลลัพธ์ของเรา คุณสามารถดูได้อย่างง่ายดายว่าผลลัพธ์จากบุคคลที่สามถูกผสมเข้ามาบ่อยแค่ไหน (ผ่านคะแนนความเป็นอิสระของเรา) และเป้าหมายของเราคือการค่อย ๆ ลดการผสมนี้ลงเมื่อเวลาผ่านไป

Google Fallback Mixing ใน Brave Search คืออะไร?
เกิดอะไรขึ้นเมื่อเปิดใช้งาน?

Brave Search กำลังปรับปรุงและพัฒนาดัชนีของตัวเองอย่างต่อเนื่อง สำหรับคำค้นหาใด ๆ ที่เราอาจไม่สามารถให้ผลลัพธ์เพียงพอ คุณสามารถอนุญาตให้เบราว์เซอร์ iBrowe ตรวจสอบ Google แบบไม่ระบุตัวตนสำหรับคำค้นหาเดียวกันได้ ฟีเจอร์นี้ เรียกว่า Google Fallback Mixer จะนำเสนอผลลัพธ์ทั้งหมดให้คุณพร้อมกัน และส่งผลลัพธ์ของคำค้นหากลับไปยัง Brave Search เพื่อให้เราสามารถปรับปรุงการตอบสนองในครั้งต่อไปได้โปรดทราบว่า หากคุณเปิดใช้งาน Google Fallback Mixing คุณจะไม่มีการโต้ตอบกับ Google แต่อย่างใด และคุณจะไม่เห็นโฆษณาการค้นหาของ Google การผสม fallback เป็นเพียงการตรวจสอบเครื่องมือค้นหาของ Google แบบไม่ระบุตัวตนสำหรับคำค้นหาเดียวกันกับที่คุณป้อนลงใน Brave Search เท่านั้น

ฟีเจอร์ Goggles คืออะไร และมันช่วยจำกัดการเซ็นเซอร์การค้นหาได้หรือไม่?

Goggles เป็นฟีเจอร์เบต้าของ Brave Search โดย Goggles ช่วยให้บุคคลหรือชุมชนใด ๆ สามารถสร้างชุดกฎและตัวกรองเพื่อจำกัดขอบเขตของสิ่งที่สามารถค้นหาได้ และ/หรือเปลี่ยนลำดับของผลการค้นหา ใครก็ตามสามารถเลือกใช้ Goggle หรือขยายมันไปยังมุมมองของผลการค้นหา Brave Search ของตนได้ โดยพื้นฐานแล้ว Goggles จะทำหน้าที่เป็นตัวเลือกในการจัดอันดับใหม่บนดัชนีของ Brave Searchนี่หมายความว่า แทนที่จะมีเพียงการจัดอันดับเดียว Brave Search สามารถนำเสนอตัวเลือกการจัดอันดับได้เกือบไม่จำกัด ทำให้สามารถใช้งานการค้นหาในกรณีที่อาจเจาะจงเกินไปสำหรับเครื่องมือค้นหาทั่วไปได้ แม้ว่า Brave Search จะไม่มีอคติด้านบรรณาธิการ แต่เครื่องมือค้นหาทุกตัวย่อมมีอคติโดยธรรมชาติในระดับหนึ่ง Goggles ช่วยให้ผู้ใช้สามารถต่อต้านอคติโดยธรรมชาตินี้ในอัลกอริทึมการจัดอันดับได้หากต้องการเข้าถึง Goggles เพียงแค่ทำการค้นหาที่ search.brave.com แล้วคลิกแท็บ Goggles บนหน้าผลลัพธ์ หรือคุณสามารถเยี่ยมชมหน้าหลักของ Goggles ได้อยากรู้เพิ่มเติมหรือไม่? อ่านเอกสารไวท์เปเปอร์ฉบับเต็มเกี่ยวกับ Goggles หรือเยี่ยมชมที่เก็บ GitHub เพื่อเรียนรู้วิธีสร้าง Goggles ของคุณเอง

โครงการ Web Discovery Project คืออะไร และฉันจะเข้าร่วมได้อย่างไร?

โครงการ Web Discovery Project เป็นวิธีที่รักษาความเป็นส่วนตัวให้คุณมีส่วนร่วมในการเติบโตและความเป็นอิสระของ Brave Search หากคุณเลือกเข้าร่วม คุณจะบริจาคข้อมูลบางส่วนแบบไม่ระบุตัวตนเกี่ยวกับการค้นหาและการเยี่ยมชมหน้าเว็บที่ทำภายในเบราว์เซอร์ iBrowe (รวมถึงหน้าที่เข้าถึงผ่านเครื่องมือค้นหาอื่น ๆ บางตัว แต่ไม่ใช่ทั้งหมด) ข้อมูลนี้ช่วยสร้างดัชนีอิสระของ Brave Search และช่วยให้มั่นใจว่าเราแสดงผลลัพธ์ที่เกี่ยวข้องกับคำค้นหาของคุณ โดย “ข้อมูล” เราหมายถึงคำค้นหา การคลิกผลการค้นหา URL ของหน้าเว็บที่เยี่ยมชมในเบราว์เซอร์ ระยะเวลาที่ใช้ในหน้าเหล่านั้น และข้อมูลเมตาบางส่วนเกี่ยวกับหน้าเว็บเอง เรียนรู้เพิ่มเติมโครงการ Web Discovery Project เป็นวิธีที่รักษาความเป็นส่วนตัวในการสร้างดัชนีของเรา ช่วยให้เราคงความเป็นอิสระและให้ผลลัพธ์ที่มีคุณภาพดีขึ้นและทันสมัยกว่า หากเราเพียงแค่พึ่งพา Web crawlers แบบง่าย ๆ อย่างที่ Google ทำหากต้องการเข้าร่วม ให้เปิดเมนูการตั้งค่าในเบราว์เซอร์ iBrowe เลือก "เครื่องมือค้นหา" ที่แผงด้านซ้าย จากนั้นสลับเปิด "โครงการ Web Discovery Project" คุณสามารถเลือกออกได้ทุกเมื่อ
