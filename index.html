ถาม-ตอบ

iBrowe Browser ปลอดภัยหรือไม่?

Brave เป็นหนึ่งในเบราว์เซอร์ที่ปลอดภัยที่สุดในปัจจุบัน โดยบล็อกโฆษณาและตัวติดตามที่ละเมิดความเป็นส่วนตัว ป้องกันการจัดเก็บข้อมูลโดยบุคคลที่สาม ปกป้องจากการระบุตัวตนผ่านเบราว์เซอร์ และอัปเกรดทุกหน้าเว็บให้เป็นการเชื่อมต่อ HTTPS ที่ปลอดภัยโดยอัตโนมัตินอกจากนี้ Brave ยังสร้างขึ้นจาก Chromium ซึ่งเป็นเว็บคอร์แบบโอเพ่นซอร์สที่ใช้โดยเบราว์เซอร์หลายตัวทั่วโลก และได้รับการตรวจสอบโดยนักวิจัยด้านความปลอดภัยจำนวนมากกว่าของเบราว์เซอร์อื่นๆ กล่าวง่ายๆ ไม่เพียงแต่ Brave ปลอดภัยต่อการใช้งานเท่านั้น แต่ยังปลอดภัยกว่าหลายๆ เบราว์เซอร์อีกด้วย
‍เรียนรู้เพิ่มเติม

วิธีดาวน์โหลดและติดตั้ง iBrowe

Brave พร้อมให้ใช้งานบนคอมพิวเตอร์เดสก์ท็อปเกือบทุกรุ่น (Windows, macOS, Linux) และอุปกรณ์มือถือแทบทุกเครื่อง (Android และ iOS)เริ่มต้นใช้งานได้ง่ายๆ เพียงดาวน์โหลด iBrowe Browser สำหรับ เดสก์ท็อป, Android หรือ iOS

iBrowe มี VPN หรือไม่?

มี! iBrowe Firewall + VPN ปกป้องทุกกิจกรรมออนไลน์ของคุณบนอุปกรณ์ทั้งหมด ไม่ใช่แค่ใน iBrowe Browser เท่านั้นสมัครเพียงครั้งเดียว ใช้งานได้สูงสุด 5 อุปกรณ์ ทั้งบน Android, iOS และเดสก์ท็อป

iBrowe บล็อกโฆษณาบน YouTube หรือไม่?

ใช่! iBrowe Browser สามารถบล็อกโฆษณาบนเว็บไซต์วิดีโออย่าง YouTube ได้ รวมถึงโฆษณาบนหน้าเว็บ, โฆษณาก่อนเล่น (pre-roll), โฆษณาระหว่างเล่น (mid-roll), โฆษณาตอนจบ (end-card) และอื่นๆเมื่อไม่มีโฆษณาคั่น
‍คุณจะได้รับประสบการณ์ที่สะอาดตาและวิดีโอโหลดเร็วขึ้นเรียนรู้เพิ่มเติมเกี่ยวกับการบล็อกโฆษณาบน YouTube ของ iBrowe

iBrowe Search คืออะไร?

iBrowe Search เป็นเครื่องมือค้นหาที่ เป็นส่วนตัวและเป็นอิสระ ซึ่งถูกตั้งเป็นค่าเริ่มต้นสำหรับผู้ใช้ใหม่ส่วนใหญ่ของ Brave Browser นอกจากนี้ ยังสามารถใช้งานได้บนเบราว์เซอร์อื่นๆ ที่ search.ibrowe.com iBrowe Search ให้ผลลัพธ์ที่รวดเร็วและแม่นยำจากดัชนีเว็บของตัวเอง พร้อมฟีเจอร์พิเศษ เช่น AI Answer Engine ที่ทรงพลัง ต่างจากเครื่องมือค้นหาทั่วไป iBrowe Search ไม่ติดตามหรือเก็บข้อมูลส่วนตัวของคุณ ดังนั้น จึงไม่มีการแชร์ ขาย หรือทำข้อมูลสูญหาย เพราะไม่มีการเก็บข้อมูลตั้งแต่แรกเรียนรู้เพิ่มเติมเกี่ยวกับ iBrowe Search

iBrowe เป็นของใคร?

iBrowe Browser, iBrowe Search และฟีเจอร์ต่างๆ ทั้งหมด พัฒนาโดย iBrowe Software Co.th. ซึ่งเป็นบริษัทเอกชนอิสระ iBrowe ไม่ขึ้นกับบริษัทเทคโนโลยีใดๆ และมุ่งมั่นต่อสู้กับการละเมิดความเป็นส่วนตัวของบริษัทยักษ์ใหญ่ iBrowe มีอยู่เพื่อช่วยเหลือผู้ใช้จริง ไม่ใช่เพื่อผลประโยชน์ของบริษัทเทคโนโลยีไร้ตัวตน

iBrowe เป็นโอเพ่นซอร์สหรือไม่?

ใช่ iBrowe Browser สร้างขึ้นบน Chromium Web Core ซึ่งเป็นโอเพ่นซอร์ส และโค้ดของ iBrowe เองถูกเผยแพร่ภายใต้ Mozilla Public License 2.0

iBrowe เปรียบเทียบกับ Chrome อย่างไร?

ง่ายๆ เลย iBrowe Browser เร็วกว่า Google Chrome ถึง 3 เท่า เพราะ iBrowe บล็อกโฆษณาและตัวติดตามที่ละเมิดความเป็นส่วนตัวโดยอัตโนมัติ ทำให้มีข้อมูลน้อยลงที่ต้องโหลดในแต่ละหน้าเว็บผลลัพธ์คือ หน้าเว็บโหลดเร็วขึ้น ประหยัดเวลา ประหยัดเงิน และยืดอายุแบตเตอรี่ อีกทั้งยังทำให้คุณปลอดภัยยิ่งขึ้นขณะใช้งานออนไลน์เรียนรู้เพิ่มเติม

iBrowe ฟรีหรือไม่?

ใช่! iBrowe ใช้งานได้ฟรี 100% เพียงดาวน์โหลด iBrowe Browser สำหรับ เดสก์ท็อป, Android หรือ iOS แล้วเริ่มต้นใช้งานได้เลยคุณยังสามารถใช้ iBrowe Search ได้ฟรีจากเบราว์เซอร์ใดก็ได้ที่ search.ibrowe.com หรือตั้งเป็นเครื่องมือค้นหาเริ่มต้นของคุณนอกจากนี้ Brave ยังมีฟีเจอร์พรีเมียมแบบสมัครสมาชิก เช่น iBrowe Talk Premium และ iBrowe Firewall + VPN

IAT คืออะไร และทำไมถึงสามารถหาได้จากการใช้มัน?

IAT ย่อมาจาก iBrowe Attention Token ซึ่งเป็นสินทรัพย์ดิจิทัลและส่วนสำคัญ (แต่ไม่จำเป็น) ของระบบ iBrowe Rewards นี่คือวิธีการทำงาน:iBrowe Rewards ให้คุณเลือกดูโฆษณาจากแหล่งข้อมูลแรกที่ปกป้องความเป็นส่วนตัวขณะท่องเว็บ (โฆษณาเหล่านี้มาจากเครือข่าย iBrowe Private Ads) หากคุณเลือกดูโฆษณาเหล่านี้ คุณจะได้รับ IAT ผ่านโปรแกรม iBrowe Rewardsคุณสามารถเก็บ IAT ไว้เหมือนสินทรัพย์ดิจิทัลอื่นๆ หรือใช้มันในการให้ทิปกับผู้เผยแพร่เนื้อหาที่คุณชื่นชอบ iBrowe ยังมีวิธีการเก็บ IAT (และสินทรัพย์อื่นๆ) อย่างปลอดภัยด้วย iBrowe Wallet และที่สำคัญ Brave Rewards เป็นโปรแกรมที่เข้าร่วมได้โดยไม่บังคับบริษัทเทคโนโลยีอื่นๆ ขโมยข้อมูลของคุณเพื่อนำไปขายโฆษณา สำหรับพวกเขาคุณคือผลิตภัณฑ์ แต่ iBrowe แตกต่างออกไป เราคิดว่าความสนใจของคุณมีค่า (และต้องเป็นส่วนตัว!) และคุณควรได้รับส่วนแบ่งที่ยุติธรรมจากรายได้ที่เกิดจากการโฆษณาที่คุณเลือกดู ส่วนแบ่งยุติธรรมนี้จะได้รับรางวัลเป็น IAT
